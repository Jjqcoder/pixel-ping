## 项目简介
本项目是一个聊天室项目，可实现一对一聊天。

## 技术选型
### 前端
React、Vite、Ant Design、TypeScript、Axios
### 后端
Express、TypeScript、Prisma、WebSocket (ws)、Tsoa
